# LlamaIndex

## Why LlamaIndex when you have langchain?
| Feature     | **LangChain**                                                                                                  | **LlamaIndex**                                                                                      |
| ----------- | -------------------------------------------------------------------------------------------------------------- | --------------------------------------------------------------------------------------------------- |
| **Focus**   | A **general-purpose orchestration framework** for building *LLM-powered applications* (agents, chains, tools). | A **data framework** designed to *ingest, structure, index, and query* large data sources for LLMs. |
| **Goal**    | Make it easy to connect models, tools, APIs, and prompts.                                                      | Make it easy for LLMs to **understand and retrieve from your data**.                                |                                                    |
## How do download LlamaIndex?
In your terminal window run this-
```bash
pip install llama-index
```
## Download requirements for this repository -
In your terminal window run this-
```bash 
pip install -r requirements.txt
```
